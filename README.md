# Stopwatch-Webapp
A modern stopwatch web application built using HTML, CSS, and JavaScript featuring start, pause, reset, and lap functionality with a clean animated UI.


This project is a modern stopwatch web application designed with a sleek
glassmorphism user interface and smooth background animations.

The stopwatch allows users to track time accurately with start, pause,
reset, and lap recording features. Lap times are displayed dynamically
in a scrollable list.

Features:
• Start, pause, and reset stopwatch
• Lap time recording
• Animated glassmorphism UI
• Responsive design
• Smooth background animations
• Clean and modern interface

Technologies Used:
• HTML
• CSS
• JavaScript
• Google Fonts
